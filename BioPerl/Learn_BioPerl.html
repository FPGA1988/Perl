1 安装

因为我们要用到只是其中的一部分的模块，所以我们没有按照网上的方法安装。 我们直接在CPAN上下载BioPerl然后解压后把模块扔到lib目录下，让Perl能 够找到就可以了。

1.1 学习条件

如果你想更好的理解这里的内容，最好把Intermediate Perl的前几章看一下。 这样方便理解。

2 面向对象

首先我们先来看一个例子： 

1:  use Data::Dumper;
2:  use Bio::Seq;
3:  
4:  # 下面创建一个seq_obj的对象，并给这个对象添加属性：是DNA序列。
5:  my $seq_obj = Bio::Seq->new(-seq=>"aaaaaaaaaaaaaaatatatctata",-alphabet=>"dna");
6:  print Dumper($seq_obj);

上面用new创建了一个新的对象，语法格式 

1:  模块名称Bio::Seq->方法名称new(-参数=>参数值);

BioPerl是面向对象的，上面我们创建了一个seqobj的对象，我们通过上面的程序 给这个对象赋予了两个属性，第一：-seq 也就是序列。第二：-alphabet也就是序列 的类型，我们指明了是dna。那这个对象中还有那些属性呢？在上面的例子中我们用 Data::Dumper把$seqobj打印出来，其实$seqobj是一个引用。引用了一个匿名的 hash。 

 1:  $VAR1 = bless( {
 2:                   'primary_seq' => bless( {
 3:                                             'length' => 25,
 4:                                             '_nowarnonempty' => undef,
 5:                                             'alphabet' => 'dna',
 6:                                             '_root_verbose' => 0,
 7:                                             'seq' => 'aaaaaaaaaaaaaaatatatctata'
 8:                                           }, 'Bio::PrimarySeq' ),
 9:                   '_root_verbose' => 0
10:                 }, 'Bio::Seq' );

通过上面我们打印的结果，我们可以看到这里还包含其他的属性，length等。 既然这是一个对hash的引用，那么我们获取属性的方法也就很简单了，解引用： 

 1:  use strict;
 2:  use warnings;
 3:  use Data::Dumper;
 4:  use Bio::Seq;
 5:  
 6:  # 下面创建一个seq_obj的对象，并给这个对象添加属性：是DNA序列。
 7:  my $seq_obj = Bio::Seq->new(-seq=>"aaaaaaaaaaaaaaatatatctata",-alphabet=>"dna");
 8:  print Dumper($seq_obj);
 9:  # 输出序列
10:  
11:  print $seq_obj->seq."\n";
12:  # 或者解引用的另一种方法
13:  print ${$seq_obj}{'primary_seq'}{'seq'}."\n";
14:  my @keys = keys %{$seq_obj};
15:  print Dumper(\@keys);
16:  # 输出长度
17:  print $seq_obj->length."\n";
18:  # 输出类型
19:  print $seq_obj->alphabet."\n";
20:

2.1 Bio::Seq属性

1:  -seq=>序列
2:  -display_id=>标签
3:  -desc=>描述
4:  -alphabet=>序列类型(dna,rna,protein),如果不声明，bioperl会自己猜测

3 Bio::SeqIO BioPerl的输入输出

Bio::SeqIO支持几乎所有的格式，http://www.bioperl.org/wiki/HOWTO:SeqIO 具体内容就看网址，不在粘贴过来。

3.1 读入

 1:  use strict;
 2:  use warnings;
 3:  use Data::Dumper;
 4:  use Bio::SeqIO;
 5:  
 6:  my      $out_out = "result.txt";
 7:  open  my $out, '>', $out_out or die  "Fail open $out_out\n";
 8:  
 9:  my $seqio_object=Bio::SeqIO->new(-file=>"Os01g0104900.fasta",-format=>"fasta");
10:  while(my $seq_object=$seqio_object->next_seq)
11:  {
12:          print $out Dumper($seq_object);
13:  }
14:  close  $out;
15:

首先，上面用Bio::SeqIO->new读入一个fasta的文件，然后创建一个对象， 这个对象指向这个文件。 然后fasta中的序列我们用$seqioobject->nextseq来取得，一次得到 一条序列。然后这个序列也是一个对象，并包含很多属性。 我们用Dumper打印出来，然后查看内容。 

 1:  $VAR1 = bless( {
 2:                   'primary_id' => '9311_GLEAN_10035453',
 3:                   'primary_seq' => bless( {
 4:                                             'primary_id' => '9311_GLEAN_10035453',
 5:                                             'display_id' => '9311_GLEAN_10035453',
 6:                                             'desc' => '',
 7:                                             'length' => 61,
 8:                                             'seq' => 'ATGGAGATCGTGGGCGAGTGCGAGTACCGGGGCGAGGCGGTGTTGGTGACGCCGAGCTCGG',
 9:                                             'alphabet' => 'dna'
10:                                           }, 'Bio::PrimarySeq' )
11:                 }, 'Bio::Seq' );
12:  $VAR1 = bless( {
13:                   'primary_seq' => bless( {
14:                                             'desc' => '',
15:                                             'primary_id' => 'Pa_GLEAN_10032244',
16:                                             'display_id' => 'Pa_GLEAN_10032244',
17:                                             'length' => 64,
18:                                             'seq' => 'ATGCCGCGGCGCTCCCTCTACCTCTCTAACCTGGACGACCAGCGATTCCTCCGTTTCTCCATCA',
19:                                             'alphabet' => 'dna'
20:                                           }, 'Bio::PrimarySeq' ),
21:                   'primary_id' => 'Pa_GLEAN_10032244'
22:                 }, 'Bio::Seq' );
23:

我们得到的内容就是一个个序列的对象了，和前面一样了。

3.2 输出

如果我们想把上面的内容进行输出的话，应该怎么做呢？同样 Bio::SeqIO提供了输出的功能，比如我们转换一下输出的格式。 

 1:  use strict;
 2:  use warnings;
 3:  use Data::Dumper;
 4:  use Bio::SeqIO;
 5:  
 6:  my $seqo_object=Bio::SeqIO->new(-file=>">result2.fasta",-format=>"fasta");
 7:  my $seqi_object=Bio::SeqIO->new(-file=>"Os01g0104900.fasta",-format=>"fasta");
 8:  while(my $seq_object=$seqi_object->next_seq)
 9:  {
10:          $seqo_object->write_seq($seq_object);
11:  }
12:

结果： 

1:  >9311_GLEAN_10035453
2:  ATGGAGATCGTGGGCGAGTGCGAGTACCGGGGCGAGGCGGTGTTGGTGACGCCGAGCTCG
3:  G
4:  >Pa_GLEAN_10032244
5:  ATGCCGCGGCGCTCCCTCTACCTCTCTAACCTGGACGACCAGCGATTCCTCCGTTTCTCC
6:  ATCA
7:

程序解读： 我们首先用Bio::SeqIO->new(-file=>">",format=>)等来新建一个输出文件。 然后一次遍历每一个seqobject,并用seqoobject->writeseq()输出。 这里我们要注意-file是输出，所以前面要加>。结果中，按照没60个bp一行的形式输出。 我们也可以把-format设为其他的形式，进行格式转换。 

 1:  use strict;
 2:  use warnings;
 3:  use Data::Dumper;
 4:  use Bio::SeqIO;
 5:  
 6:  my $seqo_object=Bio::SeqIO->new(-file=>">result2.fasta",-format=>"genbank");
 7:  my $seqi_object=Bio::SeqIO->new(-file=>"Os01g0104900.fasta",-format=>"fasta");
 8:  while(my $seq_object=$seqi_object->next_seq)
 9:  {
10:          $seqo_object->write_seq($seq_object);
11:  }
12:
